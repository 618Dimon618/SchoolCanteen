# Школьная столовая

Веб-приложение для управления питанием в школьной столовой.

## Установка

```bash
cd flask_app
pip install -r requirements.txt
python init_db.py
python app.py
```

Откройте: http://127.0.0.1:5000

## Тестовые аккаунты

## Функциональность

### Ученик
- Просмотр меню на текущий и будущие дни недели
- Заказ питания по абонементу (1 абонемент = 1 приём пищи)
- Докупка отдельных блюд за деньги
- Пополнение баланса
- Покупка абонементов (5 или 20 приёмов)
- Получение готовых заказов
- Указание аллергий в профиле
- Предупреждения об аллергенах в блюдах
- Просмотр истории заказов
- Уведомления о статусе заказов
- Отзывы о блюдах

### Повар
- Просмотр заказов к приготовлению
- Отметка о готовности (списание продуктов)
- Контроль остатков продуктов
- Создание заявок на закупку
- Просмотр статуса своих заявок
- Уведомления об одобрении заявок

### Администратор
- Статистика прибыли и расходов
- Просмотр всех заказов
- Одобрение/отклонение заявок на закупку
- Просмотр списка пользователей

## Меню

- Меню циклично по дням недели (Пн-Пт)
- Завтрак: 2 основных блюда, 2 фрукта, 2 напитка
- Обед: 2 супа, 2 горячих, 2 салата, 2 напитка
- Разные блюда каждый день

## Абонементы

- 5 завтраков - 500₽
- 20 завтраков - 2000₽
- 5 обедов - 750₽
- 20 обедов - 3000₽

1 абонемент = 1 полный приём пищи (по одному блюду из каждой категории)

## Аллергены

Отмечены следующие аллергены:
- Молоко
- Глютен
- Яйца
- Орехи
- Рыба
- Цитрусовые

Блюда с аллергенами пользователя подсвечиваются красным.
