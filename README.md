## 1. Краткое описание проекта

Разработанное веб-приложение представляет собой автоматизированную информационную систему для управления процессами питания в школьной столовой. Программный продукт решает задачи автоматизации заказов, складского учета продуктов, контроля финансов и формирования отчетности.

Система поддерживает разграничение прав доступа для трех ролей пользователей:
*   **Ученик:** осуществляет просмотр меню, оформление заказов (с оплатой с баланса или по абонементу), контроль статуса готовности и оставление обратной связи.
*   **Повар:** получает заказы в режиме реального времени, видит технологические карты блюд, управляет доступностью позиций меню в зависимости от остатков на складе и формирует заявки на закупку.
*   **Администратор:** осуществляет модерацию пользователей, согласование закупок и выгрузку аналитической отчетности.

## 2. Инструкция по установке и развертыванию

Приложение поставляется в виде Docker-контейнера, что обеспечивает идентичность окружения при запуске на любой операционной системе (Windows, Linux, macOS).

**Системные требования:**
*   Установленное ПО Docker Desktop.

**Порядок действий:**

1.  Клонируйте репозиторий проекта:
    ```bash
    git clone https://github.com/618Dimon618/SchoolCanteen.git
    ```

2.  Перейдите в директорию проекта:
    ```bash
    cd SchoolCanteen
    ```

3.  Запустите сборку и старт контейнеров:
    ```bash
    docker-compose up --build
    ```

4.  После успешного запуска веб-интерфейс приложения будет доступен в браузере по адресу:
    `http://localhost:5000`

## 4. Стек используемых технологий

*   Язык программирования: Python 3.11
*   Веб-фреймворк: Flask
*   СУБД: SQLite (с использованием SQLAlchemy ORM)
*   Фронтенд: HTML, CSS (адаптивная верстка)
*   Генерация отчетов: OpenPyXL
*   Контейнеризация: Docker
